# 爆款准则 / 招式库

> 从拆解真实爆款沉淀的传播力招式。生成卡片和正文时，本文件整段注入 LLM prompt。
> 可随时手动编辑增删，或在 Web「爆款研究 → 拆传播力」里一键入库。改动即时生效，无需重启。

## 务必遵守的基础准则

1) 封面主标题套钩子公式（痛点型 / 好奇型 / 对比型 / 数字型其一），制造一个具体冲突或反差，让人忍不住点开；绝不用平铺直叙的陈述句、内部测试名或泛泛标题。
2) 封面副标题 / 正文首段前 20 字要有钩子：要么戳中痛点，要么留悬念。
3) 收尾（最后一页 note 或正文末尾）抛一个真诚、具体的问题，引导读者评论。
4) 涉及交易 / 投资 / 理财时，弱化“量化、仓位、收益、止盈止损”等强监管词，改用“复盘、判断、自律、决策、系统、留痕”等更安全的表达以降低限流；不写收益承诺，保留风险与边界意识。

## 从拆解沉淀的传播力招式

- 标题 = 蹭当下周期 / 热点 + 人人想要的结果（如「如何在这轮 AI 行情里，搭好自己的判断系统」）。
- 开头一句话同时给“巨大回报”和“窗口正在关闭”的双钩子（回报 + 错过焦虑）。
- 正文做成编号清单，至少一条带具体数字或能算的模型，制造“值得收藏”。
- 用具体比例 / 配比制造确定感与可执行感（哪怕只是举例）。
- 收尾留一个可争论的开放点，撬动评论。
